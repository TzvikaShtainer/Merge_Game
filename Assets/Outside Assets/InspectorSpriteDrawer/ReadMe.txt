# Inspector Sprite Drawer

This asset adds a custom property drawer for Sprite fields in the Unity Inspector.

## Features:
- Previews any assigned Sprite directly inside the Inspector.
- No setup required â€“ just import the asset

## Installation:
- Import the package into your project.
- All scripts are under the 'InspectorSpriteDrawer' root folder.

## Usage:
Just use a `Sprite` field in your script and the drawer will handle it.

## Example:
```
public class Example : MonoBehaviour
{
    public Sprite icon;
}
```
